# Development Standards

## Codebase Modularity (Hard Rule)
- Max 200 lines per file where possible.
- 500-1000 lines requires refactoring.

## Admin Panel Standards
- **Sidebar Width**: Standardized at `280px` for current layout.
- **Scrollbars**: All sidebars must use themed, thin scrollbars in `auto` mode.
- **Colors**: Use `#4e73df` (primary) and `#f8f9fc` (background) for UI components.

## Environment Awareness
- Low RAM/CPU (2-core / 32GB RAM).
- Optimize for memory-heavy operations, minimize CPU spikes.
